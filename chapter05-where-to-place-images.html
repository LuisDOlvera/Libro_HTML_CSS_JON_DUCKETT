Where To Place Images In Your Code

Dónde se coloca una imagen en el código afectará la forma en que se muestra. Aquí hay tres ejemplos de colocación de imágenes que producen diferentes resultados.

  1. Antes del Parrafo:El parrafo comienza en una nueva línea después de la imágen
  2. Dentro del Inicio del Párrafo:La primera fila de texto, se alinea con la parte inferior de la imágen.
  3. En Medio de un Párrafo:La imagen es colocada entre las palabras del parrafo en el que aparece.

El lugar donde coloca la imagen en el código es importante porque los navegadores muestran elementos HTML en una de dos formas:

Los elementos de bloque siempre aparecen en una nueva línea. Los ejemplos de elementos de bloque incluyen los elementos h1 y p. Si al img le sigue un elemento de nivel de bloque (como un párrafo), el elemento de nivel de bloque se ubicará en una nueva línea después de la entrada, como se muestra en el primer ejemplo de esta página.

Los elementos en línea se ubican dentro de un elemento de nivel de bloque y no comienzan en una nueva línea. Ejemplos de elementos en línea incluyen a los elementos b, em, e img.

Si el elemento img está dentro de un elemento de nivel de bloque, cualquier texto u otros elementos en línea fluirán alrededor de la imagen, como se muestra en el segundo y tercer ejemplo de esta página.

Código Antiguo: Alineando Imágenes Horizontalmente

Align

El atributo align se usaba comúnmente para indicar cómo otras partes de una página deberían fluir alrededor de una imagen. Se eliminó de HTML5 y los nuevos sitios web deberían usar CSS para controlar la alineación de las imágenes.

Lo he discutido aquí porque es probable que lo encuentre si mira el código anterior, y porque algunos editores visuales aún insertan este atributo cuando indica cómo se debe alinear una imagen.

El Atributo Align puede tener estos valores Horizontales:

left

Esto alinea la imagen a la izquierda (permitiendo que el texto fluya alrededor de este lado derecho)

Cuando le da al atributo de alineación un valor de izquierda, la imagen se coloca a la izquierda y el texto fluye a su alrededor. Cuando el texto fluye hasta el borde de una imagen, puede dificultar su lectura.
